## DynamicArray

Represent a dynamic array like Java (list), C# (list), C++ (vector).  

## Compile

From project root folder :  

```sh
gcc src/*.c src/utils/*.c tests/*.c tests/utils/*.c -Wall -o main
```

or   

```sh
make main
```

## Launch

```sh
./main
```

or   

```sh
make run
```

## Clean 

```sh
make clean
```
