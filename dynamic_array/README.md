# DynamicArray

Represent a dynamic array like Java (list), C# (list), C++ (vector).  

## Docker

• Build  

```sh
docker build -t <name> .
```

• Run

```sh
docker run -it --rm <name>
```

## Compile

From project root folder :  

```sh
make main
```

## Launch

```sh
make run
```

## Clean 

```sh
make clean
```
